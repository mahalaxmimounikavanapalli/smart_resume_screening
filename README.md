# Smart Resume Screening
Smart Resume Screening is a web-based application developed using Python, Flask, HTML, CSS
that helps automate the resume screening process for recruiters and companies.

## Features
- Upload resumes easily
- User-friendly web interface
- Resume management system
- Organized candidate data handling
- Fast and efficient screening process

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Project Purpose
This project is designed to reduce manual effort in recruitment by organizing and managing resumes efficiently.
It demonstrates backend development, frontend design,and real-world web application development skills.
